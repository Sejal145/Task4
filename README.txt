# Task 4: Regression Analysis - House Price Prediction

This project implements a linear regression model to predict house prices based on:
- Size (numeric)
- Location (categorical)
- Number of Rooms (numeric)

## Steps Included
1. Load and Explore dataset
2. Preprocess data (scaling, encoding)
3. Feature correlation analysis
4. Train Linear Regression model
5. Evaluate model (RMSE, RÂ²)
6. Save results and plots

## Files
- regression_analysis.py : main script
- house_prices.csv : sample dataset
- predictions.csv : model predictions
- Plots: size_distribution.png, price_distribution.png, correlation_heatmap.png, predicted_vs_actual.png

pip install pandas numpy matplotlib seaborn scikit-learn
